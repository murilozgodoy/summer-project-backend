# Backend — Luma

Este repositório contém a API responsável por toda a lógica do **Luma**, uma plataforma criada em um projeto de férias para conectar clientes a profissionais da área de beleza e facilitar o agendamento de horários.

O backend gerencia:

- Usuários (clientes e profissionais)
- Serviços cadastrados
- Horários disponíveis
- Agendamentos (criação, consulta e cancelamento)

Além disso, garante:

- Autenticação segura
- Validação de dados
- Comunicação com o banco de dados
- Estrutura e regras de negócio do sistema
